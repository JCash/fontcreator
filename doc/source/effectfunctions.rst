Effect Functions
================

.. contents::
    :local:
    :backlinks: top
    
These functions can be used for post effects::

    posteffects = [shadow]
    
    [shadow]
    type = dropshadow
    color = (0, 0, 0)
    

Each effect function has it's own parameters.
