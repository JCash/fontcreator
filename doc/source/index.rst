Welcome to the documentation for FontCreator, a Python based bitmap font generator.




Features
--------

The FontCreator has many features/selling points:

- Multiple platforms (Mac, Linux, Windows)
- Performance
- Layer support
- Color functions
- Effect functions
- Blend functions
- Post effects
- Single/Pair Kerning support
- Unicode support
- Command line support

- Texture save formats:

    - Texture formats supported by PIL (.png, .tga, .jpg etc)
    - Plugin functionality for user specified format

- Glyph save formats:

    - Built in json format
    - Plugin functionality for user specified format
